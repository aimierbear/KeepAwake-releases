# KeepAwake

**[English](README.md) · [简体中文](README.zh-CN.md) · [繁體中文](README.zh-TW.md) · [日本語](README.ja.md) · [한국어](README.ko.md)**

2 つの独立したスイッチで Mac のスリープ動作を細かく制御する macOS メニューバーアプリ — 蓋を閉じたまま Mac を起こし続ける、作業中はアイドルスリープをブロックする。

> 📦 **本リポジトリでは署名 + 公証済みの `.dmg` のみを配布しています。ソースコードはプライベートリポジトリに保存されています。**

## ⬇️ ダウンロード

最新版 → **[KeepAwake-releases/releases/latest](../../releases/latest)**

すべての `.dmg` は Developer ID `C3SNXE45AV (Xiaoneng Wu)` で署名され、Apple によって公証されています。macOS Gatekeeper は「インターネットからダウンロードされたアプリ」の二次確認ダイアログなしに受け入れます。

## 🧰 システム要件

- macOS 14 Sonoma 以降
- Apple Silicon または Intel

## ✨ 機能

- **蓋を閉じても起動維持** — 蓋を閉じた状態でも Mac を動作させ続ける（初回のみ管理者認証が必要）
- **アイドルスリープ防止** — KeepAwake 実行中はアイドル起因のスリープを抑制（IOKit assertion）
- **自動オフタイマー** — 30 分 / 1 / 2 / 4 / 8 時間から選択
- **ログイン時自動起動、前回状態の記憶、終了時にスイッチ自動リセット**
- **クラッシュ時の自動復旧** — アプリがクラッシュしても、ヘルパーが 60 秒後に `disablesleep` を 0 に戻す
- **15 言語対応** — システム言語に追従

## 📥 インストール

1. [最新リリース](../../releases/latest) から `KeepAwake-0.2.0.dmg` をダウンロード
2. DMG をダブルクリックしてマウントし、**KeepAwake.app** を `/Applications` にドラッグ
3. Launchpad またはアプリケーションフォルダから起動
4. 「蓋を閉じても起動維持」を初めてオンにすると、macOS がバックグラウンド項目の追加許可を求めます → **許可**をクリック

## ⚠️ 蓋を閉じた状態での発熱リスク — 必ずお読みください

**MacBook の主要な放熱口はキーボードの下にあります。** 蓋を閉じて机に平置きすると空気の流れが妨げられます。この状態で高負荷処理（動画書き出し、コンパイル、ML 学習、長時間の AI 推論、ゲーム）を走らせると、以下の問題が起こる可能性があります：

- CPU/GPU サーマルスロットリング → タスクがむしろ**遅くなる**
- 長時間の高温によるバッテリー劣化加速
- 極端な場合はサーマルシャットダウン、さらにはハードウェア損傷

**安全な使い方:**

- ✅ **軽負荷**のバックグラウンド処理に適しています — ファイル同期、ダウンロード、SSH / リモートデスクトップ維持、AirPlay 受信
- ⚠️ 蓋を閉じて**重負荷**タスクを実行する場合は、必ずノート PC を持ち上げるかクーリングスタンドを使用してください。ベッド、毛布、ソファ、バッグの中に**絶対に**置かないでください。
- ❌「蓋を閉じる + 重負荷 + 通気孔を塞ぐ」の 3 つを同時にしないでください

`アイドルスリープ防止` もバッテリー節約の動作を上書きします。必要なときにオン、不要になったらオフにしましょう。

**KeepAwake は温度や負荷を監視しません。リスク管理はユーザー自身の責任です。**
