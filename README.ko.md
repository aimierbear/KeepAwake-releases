# KeepAwake

**[English](README.md) · [简体中文](README.zh-CN.md) · [繁體中文](README.zh-TW.md) · [日本語](README.ja.md) · [한국어](README.ko.md) · [Español](README.es.md) · [Français](README.fr.md) · [Deutsch](README.de.md) · [Português](README.pt-BR.md) · [Italiano](README.it.md)**

두 개의 독립 스위치로 Mac의 절전 동작을 세밀하게 제어하는 macOS 메뉴 막대 앱 — 덮개를 닫은 상태에서도 Mac을 깨어 있게 유지하고, 작업 중 유휴 절전을 차단합니다.

> 📦 **본 저장소는 서명 + 공증된 `.dmg` 파일만 배포합니다. 소스 코드는 비공개 저장소에 있습니다.**

## ⬇️ 다운로드

최신 릴리스 → **[KeepAwake-releases/releases/latest](../../releases/latest)**

모든 `.dmg`는 Developer ID `C3SNXE45AV (Xiaoneng Wu)`로 서명되고 Apple에 의해 공증되었습니다. macOS Gatekeeper는 "인터넷에서 다운로드한 앱" 재확인 대화 상자 없이 바로 열어줍니다.

## 🧰 시스템 요구 사항

- macOS 14 Sonoma 이상
- Apple Silicon 또는 Intel

## ✨ 기능

- **덮개 닫기 방지** — 덮개를 닫아도 Mac이 계속 실행 (최초 1회 관리자 권한 필요)
- **유휴 절전 방지** — KeepAwake 실행 중에는 유휴로 인한 시스템 절전을 차단 (IOKit assertion)
- **자동 종료 타이머** — 30분 / 1 / 2 / 4 / 8시간 선택 가능
- **로그인 시 자동 시작, 이전 상태 기억, 종료 시 자동으로 스위치 초기화**
- **크래시 자동 복구** — 메인 앱이 크래시되면 60초 후 헬퍼가 `disablesleep`을 0으로 자동 복원
- **15개 언어 지원** — 시스템 언어에 따라 자동 전환

## 📥 설치

1. [최신 릴리스](../../releases/latest)에서 `KeepAwake-0.3.0.dmg` 다운로드
2. DMG를 더블 클릭하여 마운트하고, **KeepAwake.app**을 `/Applications`로 드래그
3. Launchpad 또는 응용 프로그램에서 실행
4. "덮개 닫기 방지"를 처음 켤 때, macOS가 백그라운드 항목 추가 권한을 요청합니다 → **허용** 클릭

## ⚠️ 덮개 닫은 상태에서의 발열 위험 — 반드시 읽어주세요

**MacBook의 주요 발열구는 키보드 아래에 있습니다.** 덮개를 닫고 노트북을 책상에 평평하게 놓으면 공기 흐름이 제한됩니다. 이 상태에서 고부하 작업(동영상 내보내기, 컴파일, ML 학습, 장시간 AI 추론, 게임)을 실행하면 다음과 같은 문제가 발생할 수 있습니다:

- CPU/GPU 서멀 스로틀링 → 작업이 오히려 **느려짐**
- 장시간 고온으로 인한 배터리 수명 급속 저하
- 극단적인 경우 강제 종료, 나아가 하드웨어 손상

**안전한 사용법:**

- ✅ **저부하** 백그라운드 작업에 적합합니다 — 파일 동기화, 다운로드, SSH / 원격 데스크톱 유지, AirPlay 수신
- ⚠️ 덮개를 닫고 **고부하** 작업을 실행할 경우 반드시 노트북을 받치거나 쿨링 스탠드를 사용하세요. 침대, 담요, 소파, 가방 안에 **절대** 두지 마세요.
- ❌ "덮개 닫기 + 고부하 + 통풍구 막힘" 세 가지를 동시에 하지 마세요

`유휴 절전 방지` 또한 배터리 절약 동작을 무시합니다. 필요할 때만 켜고, 필요 없으면 꺼주세요.

**KeepAwake는 온도나 부하를 감지하지 않습니다. 위험 관리는 사용자 본인의 책임입니다.**

### 🌡️ v0.3 신규: 수동 과열 보호 (기본 활성화)

"덮개 닫기 방지"가 켜져 있는 동안, KeepAwake는 시스템 열 상태(`NSProcessInfo.thermalState`)를 모니터링합니다:

- `Serious` 상태가 60초 지속 → "덮개 닫기 방지"를 자동으로 꺼서 시스템이 절전되어 식을 수 있도록 합니다
- `Critical` → 즉시 끕니다

발동 시 알림이 표시되고, 메뉴 상단에도 경고 배너가 나타납니다. 메뉴 하단 환경설정의 "덮개 닫을 때 과열 보호"에서 끌 수 있지만, **권장하지 않습니다**.

**주의: 소프트웨어 수동 보호는 물리적 냉각을 대체할 수 없습니다.** "덮개 닫기 + 고부하 + 통풍구 막힘"을 동시에 하지 마세요.
